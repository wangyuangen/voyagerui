# voyagerui

### 介绍
yorkovevoyager的前端实现

### 开始

- 快速生成接口请求代码

```
--后端接口项目启动后,在前端项目的/gen目录下,通过node命令快速生成接口请求的typescript代码

node gen-api.js

```
